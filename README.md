Inanimate Divorce Court
A courtroom drama web app where household objects file for divorce and Judge Microwave rules on their fate.
What It Is
Single-file HTML app with 10 pre-loaded divorce cases. Each case reveals in 3 stages:
VS Screen → Arguments → Verdict
Cases include Spoon vs Fork, Six vs Seven (cannibalism), Mirror vs The User, and 7 more.
How to Use

Open index.html in a browser — no install, no server needed
Click REVEAL ARGUMENTS to show both sides
Click HEAR THE VERDICT to get Judge Microwave's ruling
Click NEXT CASE to proceed
Use 📝 File a Divorce to submit a custom case via the modal

What's Inside

10 hardcoded cases as a JS array
CSS transitions for staged reveals (arguments + verdict fade in on class toggle)
Animated ticker, floating emojis, progress dots
Modal form with a deliberately useless submit (the court is on lunch break)
Fully responsive, no dependencies, no framework
